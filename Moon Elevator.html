Elevator To The Moon

Description

The elevator to the moon is a contraption that pretty much does exactly what it is, takes you to the moon through an elevator. Instead of using a spaceship or a rocket, passengers will be able to use a single one-ride elevator all the way to the moon. For only a seven day ride, riders will be able to go up about 60,000 miles into the atmosphere.
